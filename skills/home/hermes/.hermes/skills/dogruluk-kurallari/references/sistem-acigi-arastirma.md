## Adım 3: Süzgeç — "ChatGPT bunu da söyler mi?"

Her bulduğun fırsat için:
1. "ChatGPT'ye sorsam aynı cevabı alır mıyım?" → EVET ise sil
2. "Bilal'in stack'ine özel mi?" → HAYIR ise sil
3. "Sıfır insan teması mümkün mü?" → HAYIR ise sil
4. "Kurulum gerekiyor mu?" → EVET ise "minimum kurulum" kabul et, "yok gibi" değil
5. "Dolar kazandırır mı?" → HAYIR ise ikincil planda tut

## Adım 4: Sistem Açığını Formüle Et

Formül: `(Bilal'in stack'inin yapabildiği) - (piyasada olan) = sistem açığı`

**Örnek:**
- Bilal: n8n + Crawl4AI + IG API + Hyperframes ile otomatik içerik fabrikası kurabilir
- Piyasa: Gumroad'da n8n template satanlar var ama Crawl4AI + Hyperframes İÇEREN sıfır
- **Açık:** Crawl4AI + Hyperframes içeren n8n workflow paketi

## Adım 5: Fiyatlandırma ve Gelir Tahmini

Data'ya dayalı konuş:
- Gumroad'da tek workflow 4-10$ (AI SEO Agent), bundle 10-15$
- R10'da yazılım 5.000-15.000₺

**Tahmin değil, data göster:**
- "X satıcısı Y$'dan Z adet satmış" → Bunun üzerinden konuş
- "Bilmiyorum, kontrol edeyim" → Daha fazla data topla
