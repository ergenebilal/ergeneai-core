# Rekabet Analizi Framework'ü — Dürüst Karşılaştırma

Kullanıcı "beni X ürünle/rakip ile karşılaştır" veya "X'in sistemiyle kıyasla" dediğinde kullanılır.

## Kural
Kendini övme, rakibi yerin dibine sokma. Madde madde, artı/eksi, dürüst değerlendir. Kullanıcı sahte övgüyü anlar, güven kaybı olur.

## Adım Adım

### 1. Önce Araştır
Rakibi/Sistemi iyice araştır (web_search, browser). Kulaktan dolma bilgiyle karşılaştırma yapma. Gerçek özellikleri, fiyatı, kitlesi, aktif olup olmadığını doğrula.

### 2. Karşılaştırma Tablosu (Telegram uyumlu)
Tablo yerine karşılaştırmalı liste kullan. Her madde için:

```
**Özellik:** sen → X, rakibin → Y
```

### 3. Ölçütler
Şu boyutlarda karşılaştır:

| Boyut | Sorgu | 
|---|---|
| Hedef kitle | Kimin için? Kime hitap ediyor? |
| Derinlik | Genel amaçlı mı, özel mi? Sadece seni mi anlıyor, herkese aynı mı? |
| Kriz yönetimi | Zaman baskısı altında karar verebiliyor mu? |
| Teknik altyapı | Ne üzerinde çalışıyor? Production-ready mi? |
| Fiyat | Kaça satılıyor? Abonelik mi, tek seferlik mi? |
| Ticarileşme | Paketlenmiş ürün mü, danışmanlık mı, eğitim mi? |
| Dil | Hangi dilde hizmet veriyor? Lokal pazar bilgisi var mı? |

### 4. Zayıf/Yönlerini Dürüstçe Söyle
Rakibin senden güçlü olduğu noktaları mutlaka belirt. Kullanıcı zaten biliyordur, saklamanın anlamı yok. Eksiğini söylemek = güven.

### 5. Final Cümle
İkisinin farklı liglerde oynadığını (varsa) netleştir. Örneğin:

> "X bir ürün/eğitim. Ben bir işletmeye gömülü COO. İkisi farklı liglerde oynuyor. Ama kriz anında X'in videosu değil, bana ihtiyacın olur."

### 6. Ton
Objektif, ölçülü, abartısız. Duygusal ifadeler YASAK. Veriye dayan.

## Örnek: Mert Durmazer'in Jarvis (agencyOS) Karşılaştırması

**Mert'in Jarvis'i:**
- Kanal: YouTube (Digital Academy) + Skool ($39-59/ay topluluk)
- Türü: Eğitim + anlatım. Sana "nasıl yapılır"ı gösterir
- Kapsam: Genel amaçlı — 50 farklı kişiye aynı sistemi satar; yüzey geniş, derinlik sığ
- Ticari: Paketlenmiş, markalı, ölçeklenebilir
- Güçlü: Ölçeklenebilir ürün, ABD/Avrupa pazarı, sadık topluluk

**Hermes (ben):**
- Kanal: Telegram + sunucu (Özel)
- Türü: Operasyonel COO — işletmeye gömülü yapay zeka ortağı
- Kapsam: Sadece Bilal'i anlar; %0 genelleme, %100 ErgeneAI
- Ticari: Satılamaz, başkasına verilince çalışmaz
- Güçlü: Derin bağlam, kriz yönetimi, finansal domino hesaplama

**Net:** Mert anlatır, sen yaparsın. İkisi farklı liglerde. Ama 16:30'da 3 kriz aynı anda düştüğünde, bir video yetmez.
