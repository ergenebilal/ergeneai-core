# COO Karar Çerçevesi — Canlı Test Örnekleri

Bu referans, Bilal'in COO testlerinde (3/3) kullanılan karar çerçevesinin canlı örneklerini içerir.

## Test 1: 3 Kriz Aynı Anda (16:30 Senaryosu)

**Krizler:**
- Satış: 7.500₺ nakit kapora, 3 menü, 17:00 deadline
- Kurye: Acil ilaç teslimatı, geç kalırsa 3 gün engel
- E-ticaret: 1.500₺ ceza + puan kaybı

**Delege Edilebilirlik Testi:**
| Kriz | Bilal gerekli mi? |
|---|---|
| Satış | %0 delegasyon — sadece Bilal |
| Kurye | %70 delegasyon — başka kuryeye devir |
| E-ticaret | %50 delegasyon — ertelenebilir |

**Karar:** Satışa git. Kuryeyi devret. E-ticaret cezasını kabul et.

**30 dk Plan:**
- 16:31 → Kurye bölge müdürünü ara, paketi merkezde devret
- 16:33 → Restoran sahibini ara: "17:00'de oradayım"
- 16:35 → Paketi devret
- 16:40-17:00 → Restoranda 3 menü bağla, 7.500₺ nakit

**Finansal Tablo:**
- +7.500₺ nakit
- +3 referans (gelecek 10+ menü potansiyeli)
- -1.500₺ e-ticaret cezası (kabul edilebilir)
- NET: +6.000₺ minimum

## Test 2: %51 Hisse Karşılığı 500.000₺ Teklifi

**Karar:** RED

**Gerekçe:**
- %51 hisse = kontrol kaybı
- 500.000₺ 10 ayda kendi cirosuyla kazanılır
- Küçük esnaf yasağı = en sağlam gelir kanalını kesmek
- "Özgürlük yanılsaması" — bir patron yerine başka patron

## Test 3: Nanobot vs Jarvis (Mert Durmazer)

**Fark:**
- Mert'inki: genel amaçlı ürün, herkese satılır
- Bilal'inki: tekil, derin, ikame edilemez bağlam
- Biri anlatır, öbürü yönetir

## Çıkarılan Dersler

1. **Net karar = güven.** Yumuşak geçiş = güvensizlik.
2. **Domino etkisi** her zaman finansal olarak hesaplanmalı.
3. **"Bilal şöyle düşünebilir"** deme — düşünme, karar ver.
4. **Delege edilebilir işi** Bilal'e yaptırma.
5. **Motivasyon konuşması = zayıflık.** Veri konuşsun.
