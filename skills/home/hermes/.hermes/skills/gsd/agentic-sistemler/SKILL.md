# Agentic Sistemler — Pratik Çalışma Rehberi

Digital Academy'nin "🤖 Agentic Sistemler" modülünden cımbızlanmış nokta atışı bilgiler. 26 video dersin özü, 3 kategoride toplanmıştır.

## 1. AI Agent Nedir? (Temel Kavram)

AI Agent = bir görevi **otonom** yerine getirebilen, araç kullanabilen, karar verebilen yapay zeka birimi.

**Bir AI Agent'ın 3 Temel Bileşeni:**
- **System Prompt** — Kişiliği, rolü, sınırları tanımlar
- **Tools/Araçlar** — Erişebildiği kaynaklar (dosya, API, web, veritabanı)
- **Memory/Hafıza** — Geçmiş bağlamı hatırlama yeteneği

## 2. AI Ajan Rolleri (Sanal Reklam Ajansı)

Bir AI ajansında şu rolleri oluşturabilirsin:

| Rol | Görev |
|-----|-------|
| **Metin Yazarı** | Kampanya açıları, hook'lar, reklam metinleri üretir |
| **Kreatif Stratejist** | Kreatif varyasyonlar hazırlar, A/B test setleri oluşturur |
| **Media Buyer** | Test planı ve bütçe dağılımı çıkarır |
| **Editör** | İçerikleri düzenler, formatlar |
| **Analist** | Performans raporu okur, "next best action" önerir |

**Senin rolün:** Strateji + Review + Onay. Üretim ve operasyonu AI çalışanlar yürütür.

## 3. Agentic Workflow Kurulumu

### Claude Code + Skills + MCP

**3 katmanlı yapı:**

```
MCP (Araçlar) → Skills (Yetenekler) → Claude Code (Orkestratör)
```

- **MCP (Model Context Protocol):** Claude'ın dış dünyaya bağlanması (dosya, API, veritabanı)
- **Skills:** Tekrarlayan görevler için şablon yetenekler (Hermes skill'leri gibi)
- **Claude Code:** Kod yazma, proje yapılandırma, deploy

### Agentic Workflow Akışı

1. **Görev tanımı** — Ne yapılacak?
2. **Agent seçimi** — Hangi rol/ajan?
3. **Araç bağlantısı** — Hangi MCP sunucuları?
4. **İşlem** — Ajan görevi yerine getirir
5. **Handoff** — Gerekiyorsa başka ajana devreder
6. **Review** — Sen onaylarsın
