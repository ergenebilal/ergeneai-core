## 📋 Örnek Çıktı (3 Seviye)

**Sen yazarsın:** *"Gmail'e gelen önemli mailleri yapay zeka ile analiz edip, Airtable CRM'e kaydetmek istiyorum. n8n kullanacağım."*

**AI'nın vereceği cevap:**

- **Workflow 1 (Kolay):** Gmail → Filtre (Konuda 'Teklif' geçenler) → Airtable
- **Workflow 2 (Orta):** Gmail → ChatGPT (Önem skoru ver) → Airtable
- **Workflow 3 (İleri):** Gmail → ChatGPT → Airtable → Slack bildirim
